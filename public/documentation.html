Documentation API

The web application will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal information and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Return a list of All movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET :Title None JSON object holding data about the selected movie Ex: 
           
               {
                Title: string,
                Description: string,
                Genre: {
                   Name: string,
                   Description: string,
                  },
                Director: {
                    Name: string,
                    Bio: string,
                    Birth: string
                  },
                    Stars: string,
                    imageURL: string,
                   }
Return data about a genre (description) by name/title (e.g., 'Comedy') /movies/genre/[genrename] GET :genrename None A JSON object holding data about the selected genre. Example: 
    
                   
                        {
                          Name: 'Comedy', 
                          Description: ''
                        }
Get a list of all users /users GET None None A JSON object holding data about all users, structured like:
                            
                                id: 1,
                                Username: "Maria Manson",
                                Email: "maria.manson@gmail.com",
                                Password: "klo788",
                                Birthday: "1980-01-02",
                                favoriteMovies: [],
Allow new users to register /users POST None A JSON object holding data about the user to add. Example: 
                            
                                { 
                                    "Username": "", 
                                    "Email": "", 
                                    "Password": ""
                                    "Birthday": "", 
                                    "favoriteMovies": []            
                                }
Allow users to update their user info /users/[username] PUT :username A JSON object holding the new password. Ex: 
                            
                  { 
                    "username": "Maria Manson",
                    "password": "12589mm"
                  }
JSON object with the updated user. Ex: 
                            
                  { 
                    id: '1',
                    username: 'Maria Manson', 
                    password: "56983"
                  }
Allow existing users to close account /users/[id] DELETE :id None A message indicating the user was deleted
